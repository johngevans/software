README

This folder holds examples of how to create a basic OceanSITES file from example data in 
matlab and python.  While OS standards/attributes/naming conventions can change over time, 
this method should be current for the time it was created in 2016, with format V1.3-CF1.6.

See the Data Format Reference Manual or the OceanSITES website for more details:
http://www.oceansites.org/docs/oceansites_data_format_reference_manual.pdf
http://www.oceansites.org

MATLAB:
Navigate into this directory, open Template2OS.m, and click run.

PYTHON:
Navigate into this directory, and type "run Template2OS.py".

Questions pertaining to these scripts can be directed to nathan.anderson@noaa.gov.
